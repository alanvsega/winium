# winium

## API

### Installing
```
$ cd api
$ npm install
```

### Running
```
$ npm start
```

## Web

### Installing
```
$ cd web
$ npm install
```

### Running
```
$ npm start
```

## App

### Installing
```
$ cd app
$ npm install
$ npm install -g expo-cli
```

### Running
```
$ expo start
```

Install the Expo app and use it to scan QR Code
